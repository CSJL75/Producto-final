Método del Trapecio

El método del trapecio es una técnica numérica utilizada para estimar la integral definida de una función. Es especialmente útil cuando no es posible calcular la integral de manera analítica o cuando se trabaja con datos discretos en lugar de una función continua.

Se divide el intervalo de integración en varios segmentos y se aproxima la curva de la función dentro de cada segmento con un trapecio. Luego, se suma el área de todos los trapecios para obtener una estimación de la integral.

Recapitulando:

𝐴 ≈ 𝑇1 + 𝑇2 + 𝑇3

𝐴 ≈ [∆𝑥 ∗ (𝑓(𝑥0) + 𝑓(𝑥1))/2] + [∆𝑥 ∗ (𝑓(𝑥1) + 𝑓(𝑥2))/2] + [∆𝑥 ∗ (𝑓(𝑥2) + 𝑓(𝑥3))/2]

𝐴 ≈ ∆𝑥/2 [𝑓(𝑥0) + 𝑓(𝑥1)] + [𝑓(𝑥1) + 𝑓(𝑥2)] + [𝑓(𝑥2) + 𝑓(𝑥3)]

𝐴 ≈ ∆𝑥/2 [𝑓(𝑥0) + 2𝑓(𝑥1) + 2𝑓(𝑥2) + 𝑓(𝑥3)]

𝑓(𝑥)𝑑𝑥 ≈ [𝑓(𝑥0) + 2𝑓(𝑥1) + 2𝑓(𝑥2) + ⋯ + 2𝑓(𝑥𝑛−1) + 𝑓(𝑥𝑛)]

Método de Simpson 1/3

El método de Simpson 1/3 es otra técnica para aproximar integrales definidas. Esta siendo una extensión del método del trapecio y proporciona una aproximación más precisa de la integral al utilizar polinomios de segundo grado (parábolas) para ajustarse a la función en lugar de líneas rectas.

El área bajo la parábola se obtiene de la siguiente manera:

Área total:

𝐴 = ∆𝑥/3 [𝑓(𝑥0) + 4𝑓(𝑥1) + 2𝑓(𝑥2) + 4𝑓(𝑥3) + 2𝑓(𝑥4) + 4𝑓(𝑥5) + 𝑓(𝑥6)]

--------
